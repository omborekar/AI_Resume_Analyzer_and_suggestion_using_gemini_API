Cognizant ACE Program — Technical Role (12 LPA Fresher)
========================================================

ABOUT THE ROLE
Cognizant is hiring for GenC Next and ACE (Advanced Cognizant Engineering) programs
targeting final-year / recently graduated engineers with demonstrated skills in
modern software development, AI/ML, and cloud technologies.

ROLE: Associate Software Engineer — GenAI & Full Stack
Compensation: Up to INR 12,00,000 per annum (12 LPA)
Location: Pan India (Bangalore, Hyderabad, Pune, Chennai, Mumbai, Kolkata)

KEY RESPONSIBILITIES
- Design and develop scalable full-stack applications using React (frontend)
  and Python / Java / .NET (backend).
- Implement Generative AI solutions using LLMs such as Google Gemini, GPT-4,
  and Claude for intelligent automation workflows.
- Build and maintain Retrieval-Augmented Generation (RAG) pipelines using
  vector databases (ChromaDB, Pinecone, FAISS) and embedding models.
- Architect multi-agent AI systems using LangChain, AutoGen, or CrewAI
  frameworks for complex task orchestration and agentic workflows.
- Apply Prompt Engineering techniques: chain-of-thought, few-shot learning,
  structured output, ReAct patterns to improve LLM reliability.
- Deploy applications on cloud platforms (GCP, AWS, Azure) using Docker,
  Kubernetes, and CI/CD pipelines.
- Solve algorithmic problems using Data Structures and Algorithms (DSA)
  in Java, Python, or .NET.
- Collaborate in Agile/Scrum teams; write clean, testable, maintainable code.

  - Embedding models: sentence-transformers, OpenAI embeddings, Gemini embeddings

Cloud & DevOps:

Tools & Practices:

SELECTION PROCESS
  1. Online Aptitude Test (Quantitative, Verbal, Logical Reasoning)
  2. Communication Assessment (spoken English, business writing)
  3. Technical Assessment (DSA coding round — 3 problems in Java/Python/.NET)
  4. Technical Interview (system design, project discussion, AI/ML concepts)
  5. HR Interview

INTERVIEW FOCUS AREAS
  DSA Coding Round:
    - Arrays, Strings, Linked Lists, Stacks, Queues
    - Trees (BST, AVL), Graphs (BFS, DFS, Dijkstra)

    - Sorting and Searching (Merge Sort, Quick Sort, Binary Search)
    - Complexity analysis: Big-O notation

  Technical Interview:
    - Explain your Gen AI project end-to-end

    - Explain the difference between embeddings and fine-tuning
    - REST API design principles
    - React component lifecycle, hooks, state management
    - Docker containerisation and deployment
    - Cloud deployment options and trade-offs
